# General Shared Reporting Instructions

Use this folder for notes intended to be shared across models.

## Use this when

- A task spans multiple models
- Handoff context is needed
- Decisions or conventions should be centrally documented

## File location and naming

- Store shared notes in `reports/`
- Use filename format: `YYYY-MM-DD-brief-topic.md`

## Suggested sections

- Objective
- Context
- Actions Taken
- Decisions Made
- Open Questions
- Next Steps
