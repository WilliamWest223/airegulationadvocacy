# Init Logging Hub

This folder is the coordination and reporting hub for AI model activity in this repository.

## Folders

- `claude/`: Claude-specific notes and reports
- `gemini/`: Gemini-specific notes and reports
- `cursor/`: Cursor agent-specific notes and reports
- `general/`: shared notes intended for cross-model coordination

## Reporting rule

Whenever work is performed in this repository, add a markdown report in the relevant model folder:

- `init/<model>/reports/YYYY-MM-DD-brief-topic.md`

For shared or cross-model notes, use:

- `init/general/reports/YYYY-MM-DD-brief-topic.md`
